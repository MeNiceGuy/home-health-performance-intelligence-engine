Instructions to run the Home Health Innovation Decision Engine web dashboard locally:

1. Install Required Packages:
Run this command in PowerShell:

python -m pip install --user fastapi uvicorn jinja2

2. Navigate to the web_dashboard folder inside your project directory.

3. Set the OpenAI API Key in PowerShell:
$env:OPENAI_API_KEY="your_api_key_here"

4. Start the FastAPI server with Uvicorn:
uvicorn server:app --reload --host 0.0.0.0 --port 8000

5. Open your web browser and go to:
http://localhost:8000/

6. Use the web UI to upload an agency JSON file and select output format (Markdown or PDF).
The report will be generated and available for download.
